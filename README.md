# Product Management App

A full-stack Product Management mobile application built with React Native, Node.js, Express, and MongoDB.

## Features

- View product list
- Add new products
- Update product details
- Delete products
- REST API backend
- MongoDB database
- React Native mobile UI

## Tech Stack

Frontend:
- React Native
- Expo
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Backend

cd backend  
npm install  
node server.js  

### Mobile

cd mobile  
npm install  
npx expo start  

## API Endpoints

GET /api/products  
POST /api/products  
PUT /api/products/:id  
DELETE /api/products/:id
